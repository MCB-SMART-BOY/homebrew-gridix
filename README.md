# Homebrew Tap for Gridix

This is the official Homebrew tap for [Gridix](https://github.com/MCB-SMART-BOY/Gridix).

## Installation

```bash
brew tap MCB-SMART-BOY/gridix
brew install gridix
```

## About Gridix

A fast, secure, cross-platform database management tool with Helix/Vim keybindings.

- Supports SQLite, PostgreSQL, MySQL
- SSH tunnel & SSL/TLS encryption
- 19 built-in themes
- Keyboard-driven workflow
